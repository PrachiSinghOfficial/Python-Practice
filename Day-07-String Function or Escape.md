# 📖 Theory

## String

A string is a sequence of characters enclosed in single, double, or triple quotes.

Example:

```python
name = "Prachi"
```

---

## String Functions

Functions learned:

- len()
- upper()

- replace()

## String Slicing

Examples:

```python
name[0:4]
name[:5]
name[2:]
name[-1]
name[-5:]
```

---

## Escape Sequence Characters

Examples:

```python

\t

```
