# 📖 Theory

## 1. User Input

The `input()` function is used to take input from the user.

Example:

```python
name = input("Enter your name: ")
print(name)
```

---

## 2. Type Conversion

By default, the `input()` function returns data as a string.

To perform mathematical operations, we need to convert the input into another data type.

### String to Integer

```python
age = int(input("Enter your age: "))
print(age)
```

---

### String to Float

```python
price = float(input("Enter price: "))
print(price)
```

---

### Integer to String

```python
num = 25
print(str(num))
```

---

### Float to Integer

```python
num = 12.8
print(int(num))
```

# 💻 Practice Questions

### ✅ 1. Add Two Numbers

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
print("Sum =", a + b)
```

---

### ✅ 2. Find the Remainder

```python
a = int(input("Enter a number: "))
b = int(input("Enter another number: "))
print("Remainder =", a % b)
```

---

### ✅ 3. Check the Data Type

```python
name = input("Enter something: ")
print(type(name))
```

---

### ✅ 4. Comparison Operator

```python
a = 34
b = 80
print(a > b)
```

---

### ✅ 5. Average of Two Numbers

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

average = (a + b) / 2

print("Average =", average)
```

---

### ✅ 6. Square of a Number

```python
num = int(input("Enter a number: "))
print("Square =", num ** 2)
```
