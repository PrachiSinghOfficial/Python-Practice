# 📖 Theory

## 1. String

A string is a sequence of characters enclosed in single quotes (' '), double quotes (" "), or triple quotes (""" """).

Example:

```python
name = "Prachi"
```

---

## 2. String Indexing

Each character in a string has an index.

Positive indexing starts from **0**.

Example:

```python
name = "Python"

P  y  t  h  o  n
0  1  2  3  4  5
```

---

## 3. Negative Indexing

Negative indexing starts from the end of the string.

Example:

```python
name = "Python"

P  y  t  h  o  n
-6 -5 -4 -3 -2 -1
```

---

## 4. String Slicing

Slicing is used to extract a part of a string.

Syntax:

```python
string[start:end]
```

Example:

```python
name = "Python"

print(name[0:3])

Output:
Pyt
```

---

## 5. Positive Slicing

Uses positive indexes.

Example:

```python
name[2:5]
```

Output:

```text
tho
```

---

## 6. Negative Slicing

Uses negative indexes.

Example:

```python
name[-4:-1]
```

Output:

```text
tho
```

---

## 7. Between Index Slicing

Extracts characters between two indexes.

Example:

```python
name[1:5]
```

Output:

```text
ytho
```
