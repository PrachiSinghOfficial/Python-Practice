# 📖 Theory

## What is PIP?

PIP is Python's package manager. It is used to install external libraries.

Example:

```bash
pip install numpy
```

---

## What is REPL?

REPL stands for:

- Read
- Evaluate
- Print
- Loop

It executes Python code line by line.
